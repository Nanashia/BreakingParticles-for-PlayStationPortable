パーティクル崩し（以下”原作”）
http://wonderfl.net/code/d0dcc2f2d0f8012e42785ef71dab7da2abd5453d/

のPSP移植版です。

インスコ：

0. 希望があればmake。（一応ビルド済みです。）

1. ./particleを各位GAMEフォルダへ放り込む。

2. 起動

操作：

左右キー - バーの移動
LRトリガー - バーの移動を高速にする

START - ポーズ（本来SS用に作ったものですがせっかくなのでそのままに。）
SELECT - デバッグ情報の表示切り替え。
	上から残りのブロック数、落下中のパーティクル数、ボールになったパーティクル数、FPSです。
